This file demonstrates how to use the ASEngine client.

The main function in this file shows three different ways to interact ASEngine project with the ASEngine client.

  1. send_text: Send a text message to ASEngine project according to the route.

  2. send_images: Send a image message to ASEngine project according to the route.

  3. send_audio: Open the audio recorder and send the recording audio stream to ASEngine project according to the route.
