ASEngine-Client documentation¶

Welcome to ASEngine-Client’s documentation!

This project allows you to interact with ASEngine projects with the ASEngine client. Below are the steps to install and use this project.

Installation¶

To install ASEngine-Client, make sure you have the required dependencies by running the following command:

pip install -r requirements.txt

Files¶

  • example.py: This file demonstrates how to use the ASEngine client.

  • client/client.py: The main client for connecting and interacting with the ASEngine service.
